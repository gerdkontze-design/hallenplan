# Turnhallenbelegungsplan

Eine kleine React-App, um die Nutzung einer Turnhalle von Montag bis Freitag zwischen 16:00 und 22:00 Uhr zu planen.

## Funktionen

- Sportgruppen verwalten
- Wochenplan (Mo-Fr)
- Zeitbuchungen eintragen (60 Minuten oder 90 Minuten)
- Lokale Speicherung im Browser (localStorage)

## Start

1. `npm install`
2. `npm run dev`
3. Öffne `http://localhost:5173`

## Projektstruktur

- `src/App.js` – Hauptkomponente mit Buchungslogik
- `src/main.js` – Einstiegspunkt
- `src/index.css` – Tailwind-Styles
- `tailwind.config.js` – Tailwind-Konfiguration
- `postcss.config.js` – PostCSS-Konfiguration
